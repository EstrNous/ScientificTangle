# Промпт проверки качества

Проверь решение перед завершением:

- Соответствует ли оно `docs/nauchny_klubok_top1_tz.md`?
- Не добавляет ли оно фичи вне текущей задачи?
- Не ломает ли MVP end-to-end сценарий?
- Есть ли источник/provenance для фактов, если задача касается доменных знаний?
- Нет ли новых комментариев в коде без обязательной внешней причины?
- Ведется ли новая документация на русском языке?
- Обновлен ли `docs/agent_context/project_structure.md`, если изменилась структура проекта?
- Синхронизированы ли `AGENTS.md`, адаптеры агентов и `docs/agent_prompts/`, если изменились общие правила?
- Не изменены ли `README.md`, shared contracts, public API, security rules или миграции без явного запроса?
- Выполнены ли релевантные тесты, сборка или smoke-test?
- Готов ли коммит в формате `feat: сделано то-то`?
